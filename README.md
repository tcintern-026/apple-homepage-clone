# Apple Homepage Clone

## Project Overview

This project is a responsive recreation of the Apple homepage using only HTML5 and CSS3. The goal of this project was to practice building real-world webpage layouts while improving my understanding of modern CSS techniques.

## Technologies Used

* HTML5
* CSS3

## Features

* Responsive navigation bar
* Hero section
* Three product showcase sections
* Six feature cards using CSS Grid
* Newsletter section
* Footer with multiple columns
* Responsive layout using media queries
* Hover effects and smooth transitions

## Why I Used Flexbox

I used Flexbox to:

* Build the navigation bar
* Align content inside the hero section
* Arrange buttons
* Center images and text
* Create flexible layouts that adapt to different screen sizes

Flexbox made it easy to align and distribute elements efficiently.

## Why I Used CSS Grid

I used CSS Grid for the feature section because it provides an organized way to create rows and columns. The grid automatically changes from two columns on larger screens to one column on smaller devices, making the layout responsive.

## Responsive Design

Media queries were used to:

* Adjust font sizes
* Convert the feature grid into a single-column layout on smaller screens
* Improve spacing for mobile devices
* Display a mobile navigation menu icon

## What I Learned

Through this project I learned:

* The difference between Flexbox and CSS Grid
* When to use Flexbox and when to use Grid
* How to build responsive webpages with media queries
* How to organize HTML and CSS for larger projects
* How hover effects and transitions improve user experience
* How to recreate a professional landing page using only HTML and CSS
